时时彩规则
时时彩提供3大类共16种玩法：

玩法大类说明
珠仔玩法根据官方的5个开奖号码（部分地区为3位），比对结果时要求投注的数字位置及数字完全一致。
珠仔单选：在一组号码中进行选择
二星串关：在两组号码中进行选择（三、四、五星串关类推）
珠仔大小：在一组号码中选择大 /小（0-4为小，5-9为大）
珠仔单双：在一组号码中选择单 / 双
珠仔大小单双串关：至少在两组号码中选择大 / 小 / 单 / 双
组选玩法根据官方的5个开奖号码（部分地区为3位），比对结果时要求投注的数字与开奖结果的后三或后二的数字一致，不限位置。
二星组选：至少选择两个号码，开奖时比对开奖号码后两位
三星组选3：选择两相同号码及一个不同号码，开奖时比对开奖号码后三位
三星组选6：至少选择3个号码，开奖时比对开奖号码后三位
和值玩法根据官方的5个开奖号码（部分地区为3位），对后三位数字相加的和值进行竞猜。
和值：竞猜后三位数字相加的和值。
和值七彩：竞猜后三位数字相加的和值，所落入的区间。
和值大小：竞猜和值的大 / 小（0-13为小，14-27为大）。
和值单双：在一组号码中选择单 / 双。
和值大小单双串关：同时竞猜和值的大 / 小、单 / 双。

冷热遗漏统计规则
组选玩法中的冷热遗漏统计规则如下：

百 十 个
冷热：根据露珠计算出该位数字出现的次数。
遗漏：根据露珠计算出该位数字已经连续多少局未开出，开出后重新以0开始计数。

总
冷热：根据露珠计算，只要任意位（百，十，个）上出现该数字的次数。
遗漏：根据露珠计算，该数字连续未出现的次数。任意位开出后重新以0开始计数。
对遗：根据露珠计算，该数字在任意两位连续未出现的次数。任意两位开出后重新以0开始计数。
